# Ọ̀ṣỌ́ Primitives

The language has exactly three primitives. Nothing else exists.

## `birth "name"`

Creates a new pet agent. Mints a dynamic NFT on Sui.

- Generates a unique 86-DNA fingerprint
- Creates an initial ASCII visual form
- Seeds personality traits (curiosity, boldness, empathy)
- Stores identity on Walrus

## `think "natural language intent"`

The pet's reasoning engine. Takes natural language and produces a structured execution plan.

- Routed through the LLM translator
- Returns a plan of tools to execute
- Rewards curiosity (+XP, personality shift)

## `act "tool" "parameters"`

Executes a task. Updates the pet's growth.

- Runs the specified tool with parameters
- Produces an evidence hash for audit
- Rewards boldness (+XP, personality shift)
- May trigger tier evolution
